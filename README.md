# **Kiera Wilson's Portfolio**
This portfolio show cases my work as a developer. Information is included about my work experience as well as technical skils. The application leverages onceUI magic portfolio, react, next.js, and tailwind.

**1. Install dependencies**
```
npm install
```

**2. Run dev server**
```
npm run dev
```
